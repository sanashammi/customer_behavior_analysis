# 📊 Customer Behavior Analysis 

## 🚀 Project Overview

This Data Analytics project focuses on analyzing customer behavior data to uncover meaningful business insights. The project combines Python, SQL, PostgreSQL, and Power BI to perform data cleaning, analysis, and visualization. The final output includes an interactive dashboard, analytical report, and presentation summarizing key findings.

---

## 📁 Dataset Description

The dataset contains customer-related information, including:

- Customer ID
- Age Group
- Gender
- Subscription Status
- Product Category
- Purchase Amount
- Review Rating
- Shipping Type
- Previous Purchases
- Discount Applied
- Payment Method

The dataset was cleaned and transformed to ensure accuracy and consistency before analysis.

## 📊 Key Insights and Results

- Clothing generated the highest revenue among all product categories.
- Most customers were non-subscribers.
- Young adults contributed the highest sales and revenue.
- Customer review ratings remained consistently positive.
- Revenue and sales varied significantly across product categories.

## ▶️ How to Run the Project

### Clone Repository

```bash
git clone https://github.com/your-username/customer-behavior-analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### Run Analysis

```bash
jupyter notebook
```

Open the notebook and execute all cells.

### SQL Analysis

- Import dataset into PostgreSQL using pgAdmin.
- Execute queries from `queries.sql`.

### Power BI Dashboard

- Open the `.pbix` file in Power BI Desktop.
- Refresh data connections if necessary.
